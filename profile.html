Title: Gera Easy Buy -- Online Shopping Store

Gera Easy Buy is a modern online shopping platform designed to make buying everyday products fast, easy, and convenient. The store offers a wide range of items including phones, gadgets, accessories, clothes, and shoes — all in one place. Built with HTML, CSS, and basic JavaScript, the project focuses on creating a clean, responsive, and user-friendly interface that works smoothly across all devices.

The goal of Gera Easy Buy is to give users a seamless shopping experience with clear product displays, an intuitive layout, and simple navigation. Future updates will include features like a shopping cart, order calculator, and secure checkout to enhance functionality and improve user interaction.

Technologies Used: HTML & CSS

Project Goals

To develop a fully functional, responsive online store that allows customers to explore, select, and purchase products easily while showcasing practical front-end web development skills.
